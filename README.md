# Relógio Analógico + Digital

Projeto desenvolvido com base no tutorial do canal **Manual do Dev** no YouTube, com melhorias adicionadas: além do relógio analógico, foi implementado um relógio digital sincronizado.

## 🚀 Tecnologias
- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 Funcionalidades
- Relógio analógico com ponteiros de horas, minutos e segundos.
- Relógio digital em formato 24h (HH:MM:SS).
- Atualização em tempo real.

## 📂 Estrutura
```shell
/clock-project
│
├── index.html           
├── css/
│   └── style.css        
├── js/
│   └── script.js
└── README.md
```

## 🖼️ Demonstração
O relógio analógico aparece centralizado na tela, e logo abaixo o relógio digital sincronizado.
